Mit Play Deals finden Sie die neuesten App-Angebote und Rabatte für kostenpflichtige Apps. Sie können kostenpflichtige Apps finden, die jetzt kostenlos heruntergeladen werden können oder einen Rabatt bieten.

Merkmale:
‣ Klares Design mit Unterstützung für den Nachtmodus
‣ Neue Apps werden ständig hinzugefügt
‣ Apps werden auf Ablauf überprüft, sodass Sie nur gültige Apps sehen
‣ Apps nach ihren Kategorien filtern
‣ Finden Sie App-Preisangebote
‣ Teilen Sie das App-Angebot, das Sie kennen
‣ Erhalten Sie Benachrichtigungen für alle App-Angebote
‣ Fügen Sie Angebote für die App hinzu, die Sie kennen

Die tatsächlichen Preise können je nach Standort leicht variieren.

Hinweis: Dies ist kein App Store. Sie können keine APKs oder ähnliches von dieser App herunterladen. Sie erhalten lediglich den Link zur entsprechenden App.
